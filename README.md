# Ciencia Social Abierta 2026

Este es el repositorio en donde se aloja la página web del curso electivo "Ciencia Social Abierta: Herramientas para la reproducilibidad, colaboración y comunicación de la investigación social", impartido en la Facultad de Ciencias Sociales durante el primer semestre del 2026.

La página fue construida totalmente en Quarto.
